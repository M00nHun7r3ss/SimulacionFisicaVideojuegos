Instrucciones de uso
- Para moverse horizontalmente, usar las teclas W para ir hacia atrás, S para ir hacia delante, A para girar a izquierda y D para girar a derecha.
- Para disparar desde cámara, usar la tecla E. (Su función es interactuar con el terreno para recoger recolectables en zonas inaccesibles para el jugador).
- Para que el personaje dispare hacia su parte frontal, usar la tecla R.
- El Cañón enemigo disparará todo el rato, cada 3 o 6 segundos. No se puede desactivar.
- El jugador morirá si toca el agua, si es tocado por una bala enemiga o si cae de la plataforma.
